# 设备信息

## 配置介绍

* 型号：FriendlyElec NanoPi R4S
* 架构：ARMv8 Processor x 6
* 内存：4G
* 固件：[immortalwrt](https://github.com/immortalwrt/immortalwrt)
* Google Direct
